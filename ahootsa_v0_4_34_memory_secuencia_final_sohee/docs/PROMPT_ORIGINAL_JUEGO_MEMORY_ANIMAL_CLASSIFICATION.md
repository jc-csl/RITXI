Original prompt used as design reference:

Create a self-contained HTML memory-card game about animal classification. Students turn over cards to find matching pairs: an animal and its group, such as frog–amphibian or eagle–bird. Include a timer, move counter, funny animal animations when a pair is found, and a final result screen with a brief fact about each animal group. The animal groups mustn’t have the same icon as the animals.

Adaptación Ahootsa:
- Interfaz en español.
- 8 parejas / 16 cartas.
- Números encima de las cartas para control por voz.
- Puerto local 7870.
- Herramientas de robot para iniciar, elegir cartas, reiniciar y consultar estado.
