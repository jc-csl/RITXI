# Cierre positivo de robot v5.9.84

## Modos afectados

Solo se activa en chat conversacional con robot:

```text
Texto + IA + Robot
Completo
```

## Actividades afectadas

Actividades cortas o respuestas locales que terminan contexto.

No se aplica a:

```text
actividades largas
conversaciones abiertas
actividades creativas máximas
actividades que mantienen contexto
```

## Animaciones positivas aleatorias

```text
cheerful1
enthusiastic1
yes1
hello1
dance1
dance2
dance3
electric1
amazed1
```

## Texto corregido

`Texto + IA` significa respuesta por texto en pantalla, no respuesta leída por voz.
