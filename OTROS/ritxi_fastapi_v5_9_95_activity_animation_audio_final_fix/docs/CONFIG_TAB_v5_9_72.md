# Pestaña Config v5.9.72

Se añaden accesos directos a:

```text
app/config/interaction_policy.json
app/config/ritxi_constants.py
```

## interaction_policy.json

Controla:

- fichas con respuesta local simple;
- fichas/actividades que deben usar Ollama;
- defaults de modelo, creatividad y timeout.

## ritxi_constants.py

Centraliza constantes Python principales:

- modelo por defecto;
- temperatura;
- URL Ollama;
- timeouts;
- sesión;
- host/puertos base.
