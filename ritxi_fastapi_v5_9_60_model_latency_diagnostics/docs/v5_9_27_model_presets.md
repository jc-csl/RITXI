# v5.9.27 · Modelos rápidos Ollama

Se incorporan modelos preconfigurados para competencias de comunicación y lenguaje:

- `qwen3:0.6b`: rápido.
- `gemma3:1b`: equilibrado, por defecto.
- `llama3.2:1b`: alternativa rápida de Llama.
- `llama3.2:3b`: más calidad, más lento.

También se añade `app/config/model_presets.json` y se usa `app/prompts/system_prompt.txt` como bloque de instrucciones generales.
