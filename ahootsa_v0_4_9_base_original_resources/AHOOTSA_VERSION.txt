Ahootsa Realtime Ollama v0.4.9 - Base Original Resources

Objetivo:
- Nueva base estable del proyecto Ahootsa.
- Mantener intactos todos los recursos de la app oficial de Reachy Mini Conversation App.
- Usar solo herramientas oficiales para robot:
  dance, stop_dance, play_emotion, stop_emotion, camera,
  idle_do_nothing, move_head, sweep_look, remember, forget.
- Añadir únicamente ask_ollama como herramienta nueva para consultar Ollama local.
- ask_ollama NO se activa por defecto para todo.
- ask_ollama se activa solo cuando el usuario pide explícitamente IA local, Ollama, modelo local o ahootsa-local.
- No cargar ni forzar D:\RITXI\reachy-mini-emotions-library.
- No usar JSON/OGG locales.
- No llamar a media.play_sound().
- No modificar frontend, perfiles oficiales ni herramientas originales.

Instalación:
cd D:\RITXI\ahootsa_v0_4_9_base_original_resources
.\scripts\windows\08_liberar_puertos_ahootsa.ps1
.\scripts\windows\04_limpiar_variables_emociones_locales.ps1
.\scripts\windows\00_crear_modelo_ollama_ahootsa.ps1
.\scripts\windows\01_instalar_ahootsa_realtime_ollama_en_desktop.ps1
.\scripts\windows\05_instalar_metadata_desktop.ps1

Después cerrar y abrir Reachy Mini Desktop App.
